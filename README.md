## 如何使用？

本项目基于ollama环境验证，你可以参考 [基于Ollama搭建本地LLM服务](https://mp.weixin.qq.com/s/hFF8kdPo6eF7mstiVdsJQg) 完成环境搭建，并安装qwen3.5模型。

安装依赖：

```
npm install
```

然后启动即可：

```
npm run dev
```

打开浏览器访问： http://127.0.0.1:3000/
